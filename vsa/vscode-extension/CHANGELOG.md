# Change Log

All notable changes to the "vsa-vscode" extension will be documented in this file.

## [0.1.0] - 2025-11-05

### Added
- Initial release
- Real-time validation on file save
- Diagnostics for VSA violations
- Command palette integration:
  - VSA: Validate Architecture
  - VSA: Generate Feature
  - VSA: List Features
  - VSA: Generate Manifest
- Quick fixes for common issues:
  - Create missing files
  - Rename files to follow conventions
  - Generate test files
- YAML schema for vsa.yaml with auto-completion
- Status bar integration
- File watcher for vsa.yaml changes
- Configuration options:
  - validateOnSave
  - validateOnOpen
  - configPath
  - showWarnings
