feat(vsa): Complete Milestone 1 - Project Setup & Core Infrastructure

Implement the foundational architecture for the Vertical Slice Architecture
Manager (VSA), a Rust-based CLI tool for enforcing vertical slice architecture
with bounded context support.

## Core Features

### Workspace & Crates
- Create Rust workspace with 3 crates (vsa-core, vsa-cli, vsa-wasm)
- Configure cargo workspace with shared dependencies
- Setup build profiles for release optimization

### Core Library (vsa-core)
- Configuration: YAML parsing with serde, multi-language support
- Scanner: File system traversal with walkdir
- Patterns: Regex-based file type detection (Command, Event, Handler, etc.)
- Validator: Structure validation with detailed error reporting
- Manifest: JSON/YAML manifest generation
- Framework: Optional event-sourcing-platform integration

### CLI Tool (vsa-cli)
- init: Initialize VSA configuration with templates
- validate: Validate project structure against conventions
- list: List contexts and features with tree output
- manifest: Generate feature inventory

### CI/CD Pipeline
- GitHub Actions for multi-platform testing (Linux, macOS, Windows)
- Code quality checks (rustfmt, clippy)
- Code coverage with tarpaulin
- Security audit with cargo-audit
- Automated release workflow

### Documentation
- 5 comprehensive Architecture Decision Records (ADRs)
- Detailed README with quick start guide
- PROJECT-PLAN with 6 milestones
- CHANGELOG for version tracking
- Makefile for development tasks

### Testing
- 13 unit tests covering core functionality
- Tests for configuration, scanning, patterns, validation
- All tests passing on stable Rust

## Technical Stack

- **Language:** Rust 2021 Edition
- **Dependencies:** clap, serde, serde_yaml, handlebars, walkdir, regex, chrono
- **Testing:** cargo test, tarpaulin for coverage
- **Linting:** rustfmt, clippy
